# @aps/cli — APS helper CLI

This CLI provides simple helpers for working with OpenAPI specs and the local MCP mock server.

Quick start (dev):

- cd tools/cli
- npm install
- npm run dev -- call GET /pets --base http://localhost:3000

Commands:
- `aps call <METHOD> <PATH>` — call an endpoint on a running server
- `aps generate <spec>` — placeholder to generate client SDKs
- `aps mcp:start` — guidance to start the local MCP server
